## Skate Rush: Road to Victory

## Game Description

**Skate Rush: Road to Victory** is a 2D action platformer where the player controls a skateboard
rider navigating through dangerous roads filled with obstacles, enemies, and intense challenges.
The gameplay emphasizes smooth controls, precise movement, and progressive challenges,
combining obstacles, enemies, collectibles, and a final boss fight.


## Features
- Simple and Responsive Controls: Keyboard-based movement and jumping designed for smooth and easy gameplay.
- Progressive Level System :
-Level 1: The player avoids road obstacles to reach the finish line.
-Level 2: All previous obstacles remain with increased difficulty, and flying birds
are introduced as enemies. Touching a bird causes player death, but birds can be
dodged.
-Level 3 (Final Level): A boss fight where both the player and the enemy have health bar. player use knife and enemy uses fire ball. 
- Combat Mechanics:
The player can use a knife to fight the final enemy.
- Scoring and Rewards: Players unlocks levels by collecting coins  throughout
each level.




## Project Details
IDE: Visual studio 2013

Language: C,C++.

Platform : Windows PC.

Genre : 2D action adventure


## How to Run the Project

Make sure you have the following installed:
- **Visual Studio 2013**
- **MinGW Compiler** 
- **iGraphics Library** (included in this repository)


Open the project in Visual Studio 2013
- Open Visual Studio 2013.
- Go to File → Open → Project/Solution.
- Locate and select the .sln file from the cloned repository.
- Click Build → Build Solution
- Run the program by clicking Debug → Start Without Debugging


## How to Play

### **Controls**
- Movement is controlled using Arrow keys or WASD keys.
- Jumping can be performed using Space key or W key.
